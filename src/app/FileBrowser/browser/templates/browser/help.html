Welcome

For HPE support users who need to download specific HPE system files, FileSublime is a file browser that provides access to raw support files, previously only indirectly available through other support tools. Our product is a simple and direct way to retrieve raw data.

FileSublime is used to filter through specific Json files which contain certain identifying information, such as system and date.

Navigation Bar

At the top of every page there is the navigation bar. You can use this to help navigate the web app or view the help section. There is a “Systems” button that brings you to the systems page, a “Logout” button that logs the current user out and a “Help” button that brings up the help page.

Systems Page

By default, you are brought to the systems page. This lists all the systems in a table along with some identifying information. It shows a system’s system ID, associated company name and the number of files associated with that system. You can use the search bar in the upper right to filter systems and find the one you are looking for. To filter each column you can click the column header to sort in either ascending or descending order. To view the files for a system, click on the respective system’s system ID. This will bring you to the system’s file page.

Files Page

On the file page you can view all the files associated with a system along with some identifying information. At the top of the page, you can see the system you are browsing’s company name and the system ID. Below that is a table listing all of the files for that system. Each file has a file name and date. You can search for files within a system by using the search bar on the top right hand corner of the table. Once you find the file you want to download, you can click “Download” on that file’s table row.

Capacity Alert

You may notice that some cells for the free capacity are colored in different in different shades of red. This is to show how much capacity is left on a specific system. The darker red color, the closer to 0% free capacity is left on the system. The ligher the red color the closer to 30% free capacity is left on the system. Any system that has more than 30% free capacity will be colored white.
